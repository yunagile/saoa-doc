热刷新使用
```js
// eros.dev.js 中添加如下配置 开启 websocket
'socketServer': {
    'port': 8899,
    'switch': true
}
```
然后看自己 eros.native.js url.jsServer 是否还存在，目前已经采取动态ip 写入，你需要删除此项配置，重新 eros dev，build app 即可。
